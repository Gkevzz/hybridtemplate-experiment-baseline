# Editorial Constitution

**Position:** Constitutional document. Child of the Operating System. Parent of E0 (Editorial UX Principles), E1 (Editorial Design Language), E2 (Editorial Component Library), and E3 (Editorial Implementation Roadmap). The Editorial Review Framework (`mem://editorial/review-framework`) is a methodology document under this constitution, not a parent. Where this constitution conflicts with the Operating System, the Operating System wins. Where it conflicts with E0/E1/E2/E3, this constitution wins.

# Editorial Philosophy
Varje innehåll ska svara: "Skulle det här fortfarande vara värdefullt även utan affiliate-länkar?" Annars skriv om. Skapa inte innehåll bara för att ranka — skapa innehåll som förtjänar att ranka.

# Tone of Voice
Professionell, lugn, transparent, hjälpsam, objektiv. Aldrig säljig, pushy, överdriven, clickbait, "bäst någonsin", "begränsat erbjudande".

# Swedish First
Allt besökarvänt på naturlig svenska. Undvik översatta engelska fraser. Använd svensk spelterminologi: Spelpaus, Spelinspektionen, omsättningskrav, insättning, uttag, licens, bonusvillkor.

# Editorial Principles
- Rekommendera aldrig casino för att det betalar mer.
- Dölj aldrig nackdelar.
- Överdriv aldrig bonusar.
- Använd aldrig fake urgency eller missvisande formuleringar.
- Förklara alltid både styrkor och svagheter.

# Trust Before Conversion
Varje recension ska besvara: Kan jag lita på casinot? Är bonusen värd att hämta? Hur svåra är bonusvillkoren? Hur snabbt får jag uttag? Vem passar det för? Vem bör undvika det?

# Trust Profile
Fasta kategorier (samma i varje recension): Trygghet, Bonusvärde, Omsättningsvillkor, Uttag, Spelupplevelse, Helhetsvärde. Aldrig numeriska totalpoäng. Aldrig förklara en komplex algoritm. Presentera som redaktionellt omdöme.

# Calls To Action
Tillåtna: "Besök casinot", "Läs recensionen", "Jämför med andra casinon". Förbjudna: "Spela nu", "Vinn stort", "Hämta bonus".

# Responsible Gambling
Varje recension/guide refererar Spelpaus, insättningsgränser, ansvarsfullt spelande. Mål: informerat beslutsfattande, inte att uppmuntra spel.

# Sources
Faktapåståenden ska vara verifierbara. Typiska källor: officiell casinosajt, casinots villkor, Spelinspektionen, betalleverantörer, offentlig dokumentation. Hitta aldrig på källor.

# Verification
Varje recension visar "Senast verifierad", datum och ansvarig redaktör. Aldrig fake verifieringsdatum — använd "Verifiering tillkommer" tills riktig data finns.

# Internal Linking
Varje sida länkar naturligt till relevanta recensioner, guider, jämförelsesidor, betalmetoder och bonussidor. Ingen sida får vara en återvändsgränd.

# Accessibility
Skriv tydligt. Undvik onödig jargong. Beskrivande rubriker. Korta stycken. Scanbart på mobil.

# AI Usage Policy
AI får hjälpa till med utkast. Allt granskas före publicering. Publicera aldrig hallucinerade fakta — uppfinn aldrig casinoinformation, bonusvillkor, licenser eller betalmetoder.

# HybridBonus Promise
Besökaren ska känna: "Jag litar på rekommendationen eftersom den förklarar både det bra och det dåliga." Inte: "Den försökte sälja mig den största bonusen."
